# Spotify_PPE
Nom du site: CDGMusic
Action: Ecouter de la musique, ajouter dans une playlist, supprimer une musique de la playlist, voir le profil de l'utilisateur,
modification de l'abonnement.

index.html: Page d'accueil.

Rechercher_musique.php: Accéder à la bibliothéque de musique, liker ou disliker, ajouter une musique dans la bibliothèque.L'utilisateur peut rechercher une musique ou trier les musiques.

Playlist.php: Permet d'accéder aux différentes playlists. delete_playlist.php: permet de supprimer une playlist. delete_song_from_playlist.php: permet de supprimer une ou des musiques de la playlist.

connexion.html: cette page permet soit de créer un nouveau compte (login.php) ou de se connecter, si le compte existe (signup.php)

formulaire_abonnement: permet à l'utilisateur de changer son type d'abonnement.

Profil.php: Permet de voir les informations de l'utilisateur connecté.

logout.php: permet de se déconnecter.

styles_home.css + scripts_site.js: style pour le site.
styles.css + index.js : style pour la page de connexion.
